---

## 🚀 المميزات الرئيسية للموقع (Key Features)

* **🎨 هوية بصرية مستوحاة من الشعار:** لوحة ألوان متناسقة بالكامل تعتمد على تدرجات (الأزرق السماوي، الوردي الحيوي، والذهبي البراق) مع إشعاعات خلفية وتأثيرات توهج (Ambient Glow).
* **📱 متجاوب بالكامل (Fully Responsive):** تصفح مثالي وسلس عبر جميع مقاسات الشاشات (أجهزة الكمبيوتر، الأجهزة اللوحية، الهواتف الذكية).
* **🖱️ مؤشر تفاعلي مخصص (Custom Dynamic Cursor):** مؤشر ماوس دائري تفاعلي يتبع حركة المستخدم بنعومة (LERP Trails) وينبض بالذهبي عند الوقوف على الروابط والبطاقات (يتم إيقافه تلقائياً على الشاشات اللمسية لتسريع التصفح).
* **🎢 معرض صور الأبطال (Hero Gallery):** معرض أفقي تفاعلي لصور أبطال الأكاديمية يدعم السحب باللمس على الجوال وبالماوس على الكمبيوتر مع الحفاظ على أبعاد الصور الكاملة دون اقتطاع.
* **💬 آراء أولياء الأمور (Easy Scroll Carousel):** سلايدر تفاعلي سلس للغاية مدعوم بخاصية محاذاة التمرير (`Scroll Snap`) مع مزامنة ذكية للمؤشرات السفلية (Dots).
* **🎯 تتبع أقسام الموقع النشطة (Active Link Highlight):** شريط تنقل علوي زجاجي (Glassmorphic Sticky Navbar) يراقب مكان تصفح المستخدم ويقوم بتغيير لون الرابط النشط تلقائياً للذهبي.
* **🧱 هيكلة الكود النظيفة:** عزل النصوص الثابتة والترجمات في ملف مستقل ليسهل تعديل المحتوى في خطوة واحدة دون المساس بملفات التصميم.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** هيكلة صفحات الويب الدلالية والصديقة لمحركات البحث (SEO).
* **CSS3:** تنسيقات مخصصة بالكامل تدعم التموضع المرن (CSS Grid & Flexbox) ومؤثرات الحركة المتطورة والتأثيرات الزجاجية (Glassmorphism).
* **JavaScript (Vanilla):** البرمجة التفاعلية للموقع بالكامل باستخدام مكتبات المتصفح الأصلية مثل (Intersection Observer API) وتأثيرات الحركات الحسابية للجسيمات (HTML5 Canvas Particles).

## ⚙️ كيفية التشغيل محلياً (How to Run)

الموقع مبني بالكامل على تقنيات الويب الأصلية الخفيفة ولا يتطلب أي عملية بناء (Build) أو تثبيت حزم اعتمادية.

### الطرق المتاحة لتشغيل المشروع:
1. **الفتح المباشر:** قم بفتح ملف `index.html` داخل أي متصفح ويب حديث.
2. **استخدام خادم محلي (مستحسن):**
   * إذا كنت تستخدم **VS Code**، يمكنك تثبيت إضافة **Live Server** والضغط على زر `Go Live` في الأسفل لتشغيل الموقع على خادم محلي تفاعلي مع ميزة التحديث التلقائي للتغييرات.
   * أو عبر سطر الأوامر باستخدام أي أداة خادم بسيطة مثل:
     ```bash
     # باستخدام Python
     python -m http.server 8000
     
     # أو باستخدام Node.js (بعد تثبيت http-server)
     npx http-server
     ```
